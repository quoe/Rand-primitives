# Rand-primitives
Генерация случайных примитивных фигур в пространстве.  
Можно передвигаться, прыгать, пересоздавать мир, подталкивать фигуры. Можно сохранять скриншоты.  

Подойдёт художникам для тренировки.  

Unity 2018.2.11f1.
